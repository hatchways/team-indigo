Tindigo API Docs


Endpoint Description
GET /account/u/#userid Gets the account information of the given UserID.
POST /account/u Creates account with provided user information.
PUT /account/u Edits account with provided user information.
POST /account/signin Signs into account with provided username and password.
GET /post/id/#postid Gets the post information of the given PostID.
POST /post/id Creates a post under the provided username.
